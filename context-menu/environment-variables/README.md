# 右键菜单进入环境变量

在桌面或文件夹空白处右键，一键打开系统“环境变量”设置窗口。

## 功能
- 右键菜单新增“环境变量”入口
- 同时支持桌面和文件夹背景菜单
- 纯注册表方案，无需常驻进程

## 文件
- `envvars_menu.reg`：添加右键菜单
- `remove_envvars_menu.reg`：移除右键菜单

## 安装
双击运行 `envvars_menu.reg`，确认导入注册表即可。

## 卸载
双击运行 `remove_envvars_menu.reg`，移除右键菜单。

## 注意
- 修改注册表前建议备份。
- 如果未生效，可重启资源管理器或重新登录。
