# NPM 全局更新检查

一个交互式 PowerShell 脚本，提供 npm 全局包的查看、更新、安装、卸载，以及 npm 自身状态检查。

## 功能
- 列出所有全局包
- 检查可更新的包并显示当前/最新版本
- 一键更新所有过期包
- 更新 / 卸载指定包
- 安装新全局包
- 查看 npm 版本与安装路径

## 使用方法
```powershell
cd "NPM全局更新检查"
powershell.exe -File .\npm-manager.ps1
```

脚本启动后按提示输入选项：
- `1` 列出全局包
- `2` 检查可更新
- `3` 一键更新所有过期包
- `4` 更新指定包
- `5` 卸载指定包
- `6` 安装新包
- `7` 查看 npm 状态
- `0` 退出

## 注意事项
- 危险操作（更新/卸载）包含确认提示，请谨慎确认。
- 需要本机已安装 npm，并配置好环境变量。
